# Repeater ID Grid

This page tracks Ottawa repeater IDs (00–FF), showing which are deployed, reserved, or available.

Click any cell to view details. The grid is served live from [MeshMapper](https://yow.meshmapper.net/), so status updates automatically as repeaters check in.
